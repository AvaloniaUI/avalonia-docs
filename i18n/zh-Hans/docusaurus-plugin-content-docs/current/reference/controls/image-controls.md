# 图像控件

这些 _Avalonia UI_ 控件允许你在应用程序中添加图像。

:::info
如果你计划绑定一个图像源，你将需要使用一个图像转换器。要了解如何做到这一点，请参见此处。
:::

|                                                      |                                                  |   |
| ---------------------------------------------------- | ------------------------------------------------ | - |
| [DrawingImage](./drawing-image.md) | 显示矢量图像。                                   |   |
| [Image](image.md)                                    | 显示位图图像。                                   |   |
| [PathIcon](./path-icon.md)         | 通过流几何指令绘制图标。                         |   |
