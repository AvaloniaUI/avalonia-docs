# XAML 预览器

XAML 预览器可在 JetBrains Rider（通过第三方 AvaloniaRider 插件）以及 Visual Studio（通过 Avalonia for Visual Studio 插件）中使用。使用 XAML 预览器，您可以在工作时实时查看 XAML 代码的更改。我们强烈推荐使用预览器以获得最佳的开发体验。

## 启用 XAML 预览器

如果您是 IDE 开发新手，以下是如何在 Rider 和 Visual Studio 中启用 XAML 预览器的方法。

<Tabs groupId="ide">  
  <TabItem value="rider" label="Rider">
  	1. 在 Rider 中，从顶部功能区选择一个 **.axaml** 文件。
    2. 在顶部功能区的右侧，点击**编辑器和预览**。
    3. 构建您的项目。

    <div className="center" style={{maxWidth:400}}>
    <img className="center" src={RiderXamlPreviewer} alt="突出显示在 Rider 中启用 XAML 预览的按钮的屏幕截图。" />
    </div>
  </TabItem>
  <TabItem value="vs" label="Visual Studio">
    在 Visual Studio 中，XAML 预览器通常默认打开。如果没有打开，请按照以下步骤启用它。

    1. 在 Visual Studio 中，打开一个 **.axaml** 文件。
    2. 在编辑器窗口顶部点击**拆分视图**。
    3. 构建您的项目。

  </TabItem>
</Tabs>

## 检查功能

1. 打开任何 **.axaml** 文件。在本例中，我们将使用默认 Avalonia 模板中的 **MainWindow.axaml**。
2. 在底部附近找到 `<TextBlock>...</TextBlock>` XAML 标签。
3. 删除 `{Binding Greeting}`。将其更改为任何文本，例如"Nice preview!"
4. 您应该会在键入时看到文本出现在预览窗格中。

<div className="center" style={{maxWidth:400}}>
<img className="center" src={TestXamlPreviewer} alt="演示 Avalonia XAML 预览器测试的屏幕截图。" />
</div>

:::tip
如果预览器对您的更改没有响应，请尝试再次构建项目。
:::
