## .axaml

注意到您的项目目录中以 `.axaml` 结尾的文件了吗？这是 Avalonia XAML 的缩写，是 Avalonia 独有的文件扩展名，用于将 Avalonia 文件与标准 XAML 文件区分开来。

## 检查您的 XAML 预览器

如果您刚在 Visual Studio 或 JetBrains Rider 中安装了 Avalonia，并且您是 IDE 开发的新手，现在可能是检查您是否可以使用 XAML 预览器的好时机。

有关如何启用和测试预览器的信息，请参阅 [XAML 预览器页面](/docs/get-started/xaml-previewers)。

## 打开主窗口文件

在项目目录的 **Views** 文件夹中，打开文件 **MainWindow.axaml**。在本教程中，我们主要将使用这个文件。

<Image light={MainWindowFileLocation} alt="显示文件树中主窗口文件位置的屏幕截图。" position="center" maxWidth={400} cornerRadius="true" margin="20,0,50,0"/>

**MainWindow.axaml** 中几乎所有内容都位于 `<Window>...</Window>` XAML 标签之间。此标签代表 Avalonia 窗口，您的应用程序将在目标平台上在其中运行。稍后当我们学习[自定义 Avalonia 窗口](/docs/get-started/starter-tutorial/customizing-the-avalonia-window)时，我们将更详细地了解 Avalonia 窗口。
