您可以通过本节快速开始使用 Avalonia。我们将通过一个简单的教程示例，循序渐进地介绍 Avalonia 的核心概念和技术。

## 先决条件

### 安装项目模板

开始前请确保已安装 [Avalonia 项目模板](../install.md):

```bash title='Bash'
dotnet new install Avalonia.Templates
```
