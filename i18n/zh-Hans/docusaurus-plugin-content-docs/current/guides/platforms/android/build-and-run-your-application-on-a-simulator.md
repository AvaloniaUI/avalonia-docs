# 在模拟器上构建和运行您的应用程序

请确保您已按照指南[构建跨平台应用程序](../../building-cross-platform-applications)进行操作。

假设您已经创建了一个名为`HelloWorld`的项目。从命令行进入`HelloWorld.Android`目录。

要为Android构建项目，请运行以下命令。

```bash
dotnet build
```

要在模拟器中运行项目，请运行以下命令。

```bash
dotnet run
```
