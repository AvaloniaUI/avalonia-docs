## Remarks

The [Avalonia.Controls.Models.TreeDataGrid.RowHeaderCell](xref:Avalonia.Controls.Models.TreeDataGrid.RowHeaderCell) class displays the visual row number (0, 1, 2...) in a
[TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid). The displayed number always represents the visual position of the row,
remaining sequential regardless of sorting or whether the data source is flat or
hierarchical.

This cell is always read-only and cannot be edited by the user. It is designed to be used
with [Avalonia.Controls.Models.TreeDataGrid.RowHeaderColumn&lt;T&gt;](xref:Avalonia.Controls.Models.TreeDataGrid.RowHeaderColumn%601) to display row numbers.
