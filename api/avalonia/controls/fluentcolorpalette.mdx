#### Parameters

`colorIndex` int

The index of the color in the palette.
The index must be between zero and [Avalonia.Controls.IColorPalette.ColorCount](xref:Avalonia.Controls.IColorPalette.ColorCount).


`shadeIndex` int

The index of the color shade in the palette.
