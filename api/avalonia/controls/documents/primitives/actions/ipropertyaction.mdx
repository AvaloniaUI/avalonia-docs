## Remarks

This interface provides object-based access to property values,
allowing UI controls like combo boxes to work without knowing the
concrete value type. Prefer using [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction&lt;T&gt;](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction%601)
when the value type is known at compile time.

The generic interfaces [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction&lt;T&gt;](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction%601) and
[Avalonia.Controls.Documents.Primitives.Actions.IBlockPropertyAction&lt;T&gt;](xref:Avalonia.Controls.Documents.Primitives.Actions.IBlockPropertyAction%601) provide default implementations
for [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.GetValueAsObject(Avalonia.Controls.Documents.Primitives.ITextEditorHost)](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.GetValueAsObject%28Avalonia.Controls.Documents.Primitives.ITextEditorHost%29), [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.SetValueAsObject(Avalonia.Controls.Documents.Primitives.ITextEditorHost,object)](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.SetValueAsObject%28Avalonia.Controls.Documents.Primitives.ITextEditorHost%2CSystem.Object%29),
and [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.GetAvailableValuesAsObjects](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.GetAvailableValuesAsObjects) that delegate to their
typed counterparts. Only [Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.ClearValue(Avalonia.Controls.Documents.Primitives.ITextEditorHost)](xref:Avalonia.Controls.Documents.Primitives.Actions.IPropertyAction.ClearValue%28Avalonia.Controls.Documents.Primitives.ITextEditorHost%29) requires an explicit
implementation because its behavior varies per action type.
