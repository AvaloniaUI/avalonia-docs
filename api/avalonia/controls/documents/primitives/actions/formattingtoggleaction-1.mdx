## Remarks

Toggle formatting actions apply or remove a formatting property from the
current selection. When the selection is empty, the property is "springloaded"
and applied to the next typed character.

Derived classes should specify the [Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction&lt;T&gt;.Property](xref:Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction%601.Property), [Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction&lt;T&gt;.OnValue](xref:Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction%601.OnValue),
and [Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction&lt;T&gt;.OffValue](xref:Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction%601.OffValue) to define the formatting behavior.
