## Remarks

Probes the clipboard via
[Avalonia.Controls.Documents.Primitives.Components.ClipboardOperations.TryGetPlainTextSnapshotAsync(Avalonia.Input.Platform.IClipboard,System.Threading.CancellationToken)](xref:Avalonia.Controls.Documents.Primitives.Components.ClipboardOperations.TryGetPlainTextSnapshotAsync%28Avalonia.Input.Platform.IClipboard%2CSystem.Threading.CancellationToken%29): when an
in-process snapshot is present it is lowered through
`PlainTextSerializer` (Serialize → Deserialize) so the source
text is preserved exactly as our serializer emits it; otherwise the
platform `DataFormat.Text` payload is parsed straight into a
snapshot. The resulting snapshot is grafted via
`TextRange.InsertSnapshot` the same way [Avalonia.Controls.Documents.Primitives.Actions.PasteAction](xref:Avalonia.Controls.Documents.Primitives.Actions.PasteAction)
inserts rich content.
