## Remarks

Removes the [Avalonia.Controls.Documents.TableColumn](xref:Avalonia.Controls.Documents.TableColumn) and the corresponding [Avalonia.Controls.Documents.TableCell](xref:Avalonia.Controls.Documents.TableCell)
from every row. If it is the last column, the entire table is removed.
