## Remarks

Produces lossless XAML that maps directly to the FlowDocument model.
Supports all block and inline types with their full formatting properties.

Uses [System.Xml.XmlReader](xref:System.Xml.XmlReader) and [System.Xml.XmlWriter](xref:System.Xml.XmlWriter)
for parsing and generation, avoiding dependency on Avalonia's XAML runtime loader.

This serializer is thread-safe for concurrent operations.
