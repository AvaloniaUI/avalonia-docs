## Remarks

[TreeDataGridColumnHeader](xref:Avalonia.Controls.Primitives.TreeDataGridColumnHeader) is responsible for displaying column headers in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid) and
provides functionality for column resizing and sorting.

Column headers can be clicked to sort the data in the [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid), and if resizing is
enabled, they can be resized by dragging the edge of the header.

This class supports the following pseudo-classes:

- :resizable - Set when the column can be resized by the user
