### GetLineCount Method

Get the number of lines in the [TextBox](xref:Avalonia.Controls.TextBox).

```csharp
public int GetLineCount()
```

#### Returns

int


#### Value

number of lines in the [TextBox](xref:Avalonia.Controls.TextBox), or -1 if no layout information is available


#### Remarks

If Wrap == true, changing the width of the [TextBox](xref:Avalonia.Controls.TextBox) may change this value.
The value returned is the number of lines in the entire [TextBox](xref:Avalonia.Controls.TextBox), regardless of how many are
currently in view.
