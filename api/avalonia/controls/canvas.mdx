## Remarks

Unlike other [Avalonia.Controls.Panel](xref:Avalonia.Controls.Panel) implementations, the [Avalonia.Controls.Canvas](xref:Avalonia.Controls.Canvas) doesn't lay out
its children in any particular layout. Instead, the positioning of each child control is
defined by the
```csharp
Canvas.Left
```
,
```csharp
Canvas.Top
```
,
```csharp
Canvas.Right
```

and
```csharp
Canvas.Bottom
```
attached properties.
