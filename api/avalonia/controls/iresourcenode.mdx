## Remarks

The interface represents a common interface for both controls that host resources
([Avalonia.Controls.IResourceHost](xref:Avalonia.Controls.IResourceHost)) and resource providers such as [Avalonia.Controls.ResourceDictionary](xref:Avalonia.Controls.ResourceDictionary)
(see [Avalonia.Controls.IResourceProvider](xref:Avalonia.Controls.IResourceProvider)).
