### BringIntoViewOnFocusChange

Gets or sets a value that determines whether the [Avalonia.Controls.ScrollViewer](xref:Avalonia.Controls.ScrollViewer) uses a
bring-into-view scroll behavior when an item in the view gets focus.

```csharp
public System.Boolean BringIntoViewOnFocusChange { get; set; }
```

Value: true to use a behavior that brings focused items into view. false to use a behavior
that focused items do not automatically scroll into view. The default is true.
