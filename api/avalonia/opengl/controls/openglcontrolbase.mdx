## Remarks

The control automatically manages OpenGL context creation, surface setup, and cleanup.

Important: Any interaction with [Avalonia.OpenGL.GlInterface](xref:Avalonia.OpenGL.GlInterface) should only happen within the
[Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlInit(Avalonia.OpenGL.GlInterface)](xref:Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlInit%28Avalonia.OpenGL.GlInterface%29), [Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlDeinit(Avalonia.OpenGL.GlInterface)](xref:Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlDeinit%28Avalonia.OpenGL.GlInterface%29), or [Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlRender(Avalonia.OpenGL.GlInterface,System.Int32)](xref:Avalonia.OpenGL.Controls.OpenGlControlBase.OnOpenGlRender%28Avalonia.OpenGL.GlInterface%2CSystem.Int32%29) method overrides.

Avalonia ensures proper OpenGL context synchronization and makes the context current only during these method calls.
Accessing OpenGL functions outside of these methods may result in undefined behavior, crashes, or rendering corruption.
