## Remarks

This struct is similar to [Nullable&lt;T&gt;](xref:System.Nullable%601) except it also accepts reference types:
note that null is a valid value for reference types. It is also similar to
[Avalonia.Data.BindingValue&lt;T&gt;](xref:Avalonia.Data.BindingValue%601) but has only two states: "value present" and "value missing".

To create a new optional value you can:

- For a simple value, call the [Avalonia.Data.Optional&lt;T&gt;](xref:Avalonia.Data.Optional%601) constructor or use an implicit
conversion from `T`
- For an missing value, use [Avalonia.Data.Optional&lt;T&gt;.Empty](xref:Avalonia.Data.Optional%601.Empty) or simply `default`
