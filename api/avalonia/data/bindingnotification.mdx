## Remarks

This class is very similar to [Avalonia.Data.BindingValue&lt;T&gt;](xref:Avalonia.Data.BindingValue%601), but where [Avalonia.Data.BindingValue&lt;T&gt;](xref:Avalonia.Data.BindingValue%601)
is used by typed bindings, this class is used to hold binding and data validation errors in
untyped bindings. As Avalonia moves towards using typed bindings by default we may want to remove
this class.
