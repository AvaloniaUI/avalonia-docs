## Remarks

Bit positions match the OpenType specification verbatim. Bits 0..31 are
sourced from `ulCodePageRange1` and bits 32..63 from
`ulCodePageRange2`, so the full 64-bit value can be tested with a
single mask. See
[the OpenType OS/2 spec](https://learn.microsoft.com/typography/opentype/spec/os2#ulcodepagerange1-bits-031).
