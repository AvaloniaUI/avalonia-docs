#### Remarks

This value can be used to display only part of [Avalonia.Media.DrawingImage.Drawing](xref:Avalonia.Media.DrawingImage.Drawing), or to surround it with empty
space. If null, [Avalonia.Media.DrawingImage.Drawing](xref:Avalonia.Media.DrawingImage.Drawing) will provide its own viewbox.
