## Remarks

A typical usage example is a [ListBox](xref:Avalonia.Controls.ListBox) control, where [Avalonia.Metadata.InheritDataTypeFromItemsAttribute](xref:Avalonia.Metadata.InheritDataTypeFromItemsAttribute) is defined on the ItemTemplate property,
allowing the template to inherit the data type from the Items collection binding.
