### AppSandboxEnabled

If you distribute your app in App Store - it should be with sandbox enabled.
This parameter enables [Avalonia.Platform.Storage.IStorageItem.SaveBookmarkAsync](xref:Avalonia.Platform.Storage.IStorageItem.SaveBookmarkAsync) and related APIs,
as well as wrapping all storage related calls in secure context. The default value is true.
