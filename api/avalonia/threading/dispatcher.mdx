## Remarks

In Avalonia, there is usually only a single [Avalonia.Threading.Dispatcher](xref:Avalonia.Threading.Dispatcher) in the application -
the one for the UI thread, retrieved via the [Avalonia.Threading.Dispatcher.UIThread](xref:Avalonia.Threading.Dispatcher.UIThread) property.
