## Remarks

AvaloniaList is similar to [System.Collections.ObjectModel.ObservableCollection&lt;T&gt;](xref:System.Collections.ObjectModel.ObservableCollection%601)
with a few added features:

- It can be configured to notify the [Avalonia.Collections.AvaloniaList&lt;T&gt;.CollectionChanged](xref:Avalonia.Collections.AvaloniaList%601.CollectionChanged) event with a [System.Collections.Specialized.NotifyCollectionChangedAction.Remove](xref:System.Collections.Specialized.NotifyCollectionChangedAction.Remove) action instead of a [System.Collections.Specialized.NotifyCollectionChangedAction.Reset](xref:System.Collections.Specialized.NotifyCollectionChangedAction.Reset) when the list is cleared by setting [Avalonia.Collections.AvaloniaList&lt;T&gt;.ResetBehavior](xref:Avalonia.Collections.AvaloniaList%601.ResetBehavior) to [Avalonia.Collections.ResetBehavior.Remove](xref:Avalonia.Collections.ResetBehavior.Remove).
- A [Avalonia.Collections.AvaloniaList&lt;T&gt;.Validate](xref:Avalonia.Collections.AvaloniaList%601.Validate) function can be used to validate each item before insertion.
