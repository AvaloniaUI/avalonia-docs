## Remarks

[TreeDataGridCell](xref:Avalonia.Controls.Primitives.TreeDataGridCell) is the base class for cells in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid) control. It provides core
functionality for cell realization, selection, and editing. Cells are created by the
[Avalonia.Controls.Primitives.TreeDataGridElementFactory](xref:Avalonia.Controls.Primitives.TreeDataGridElementFactory) and are reused as the user scrolls through the
grid.

This class implements the following pseudo-classes:

- :editing - Set when the cell is in edit mode
- :selected - Set when the cell is selected

Cell editing can be triggered through various gestures including double-tap, F2, or single
tap, depending on the [Avalonia.Controls.BeginEditGestures](xref:Avalonia.Controls.BeginEditGestures) settings of the cell's model.
