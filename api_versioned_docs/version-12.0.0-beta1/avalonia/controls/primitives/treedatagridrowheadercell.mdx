## Remarks

[TreeDataGridRowHeaderCell](xref:Avalonia.Controls.Primitives.TreeDataGridRowHeaderCell) displays the visual row number (0, 1, 2...) in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid).
The number always represents the visual position of the row, remaining sequential
regardless of sorting or whether the data source is flat or hierarchical.

This cell type is read-only and does not support editing.
