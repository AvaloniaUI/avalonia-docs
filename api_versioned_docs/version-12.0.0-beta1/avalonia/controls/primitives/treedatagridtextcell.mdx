## Remarks

[TreeDataGridTextCell](xref:Avalonia.Controls.Primitives.TreeDataGridTextCell) is used for displaying and editing text values in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid).

This cell type provides properties to control text appearance, such as text trimming,
wrapping, and alignment. When in edit mode, it displays a [Avalonia.Controls.TextBox](xref:Avalonia.Controls.TextBox) that
allows the user to modify the text value.
