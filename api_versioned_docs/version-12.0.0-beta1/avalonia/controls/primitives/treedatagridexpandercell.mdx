## Remarks

[TreeDataGridExpanderCell](xref:Avalonia.Controls.Primitives.TreeDataGridExpanderCell) is used for displaying hierarchical data in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid).
It shows an expander button that allows users to expand or collapse child rows, and
embeds another cell (the content cell) that displays the actual data value.
