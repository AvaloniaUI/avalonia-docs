## Remarks

TreeDataGridPresenterBase implements virtualization, for [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid)'s column and row
presenters.

This class handles:

- Element creation and recycling using a [Avalonia.Controls.Primitives.TreeDataGridElementFactory](xref:Avalonia.Controls.Primitives.TreeDataGridElementFactory)
- Viewport tracking and efficient scrolling
- Item change notifications
- Focused element tracking
- Bringing items into view

Derived classes must implement abstract methods to handle item-specific realization,
un-realization, and element index updates. They must also specify the orientation
in which items are laid out.
