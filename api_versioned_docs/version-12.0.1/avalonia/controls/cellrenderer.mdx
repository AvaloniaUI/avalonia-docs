## Remarks

When a column has a [Avalonia.Controls.CellRenderer](xref:Avalonia.Controls.CellRenderer), the [Avalonia.Controls.Primitives.TreeDataGridCellsPresenter](xref:Avalonia.Controls.Primitives.TreeDataGridCellsPresenter)
renders cells directly rather than creating individual control elements for each cell. This can
significantly improve performance for columns that display simple content such as text.

This is an abstract class rather than an interface so that methods with defaults can be added
in the future without breaking existing implementations.
