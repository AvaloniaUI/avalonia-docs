## Remarks

Superscript and subscript are mutually exclusive: both toggle the single
[Avalonia.Controls.Documents.RichInline.BaselineAlignmentProperty](xref:Avalonia.Controls.Documents.RichInline.BaselineAlignmentProperty), so enabling one replaces
the other automatically (there is only one model/springload key). All toggle,
springload, and toolbar-state behaviour — including lighting up at a collapsed
caret parked inside an existing superscript span — is inherited from
[Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction&lt;T&gt;](xref:Avalonia.Controls.Documents.Primitives.Actions.FormattingToggleAction%601), the same path as Bold/Italic/Underline.
