## Remarks

Margin controls the spacing outside the block element's border.
This action sets uniform margins on all four sides (top, right, bottom, left).

Common margin values:

- 0 - No margin (flush with container)
- 5 - Minimal spacing
- 10 - Standard paragraph spacing
- 15 - Moderate spacing
- 20 - Large spacing for visual separation

When applied to a selection spanning multiple paragraphs, all affected paragraphs
will have their margins set to the specified value.

Note: This action sets uniform margins only. For non-uniform margins
(different values for each side), use a dedicated margin editor panel.
