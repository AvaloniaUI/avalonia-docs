## Remarks

Two static instances ([Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.Before](xref:Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.Before) and [Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.After](xref:Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.After)) are provided
for convenience and registered in [Avalonia.Controls.Documents.Primitives.Actions.EditorActions](xref:Avalonia.Controls.Documents.Primitives.Actions.EditorActions).

[Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.CanExecute(Avalonia.Controls.Documents.Primitives.ITextEditorHost)](xref:Avalonia.Controls.Documents.Primitives.Actions.InsertTableRowAction.CanExecute%28Avalonia.Controls.Documents.Primitives.ITextEditorHost%29) returns true only when the caret is inside a [Avalonia.Controls.Documents.TableRow](xref:Avalonia.Controls.Documents.TableRow).
The new row has the same number of cells as the reference row, each containing an empty paragraph.
