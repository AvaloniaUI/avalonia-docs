## Remarks

Text alignment is a block-level property that determines how text is aligned
within a paragraph or section. Available options are:

- [Avalonia.Media.TextAlignment.Left](xref:Avalonia.Media.TextAlignment.Left) - Align text to the left edge
- [Avalonia.Media.TextAlignment.Center](xref:Avalonia.Media.TextAlignment.Center) - Center text
- [Avalonia.Media.TextAlignment.Right](xref:Avalonia.Media.TextAlignment.Right) - Align text to the right edge
- [Avalonia.Media.TextAlignment.Justify](xref:Avalonia.Media.TextAlignment.Justify) - Justify text (stretch to fill width)

When applied to a selection spanning multiple paragraphs, all affected paragraphs
will have their alignment set to the specified value.
