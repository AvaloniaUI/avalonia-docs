## Remarks

Two static instances ([Avalonia.Controls.Documents.Primitives.Actions.InsertTableColumnAction.Before](xref:Avalonia.Controls.Documents.Primitives.Actions.InsertTableColumnAction.Before) and [Avalonia.Controls.Documents.Primitives.Actions.InsertTableColumnAction.After](xref:Avalonia.Controls.Documents.Primitives.Actions.InsertTableColumnAction.After)) are provided.
Each inserts a [Avalonia.Controls.Documents.TableColumn](xref:Avalonia.Controls.Documents.TableColumn) and an empty [Avalonia.Controls.Documents.TableCell](xref:Avalonia.Controls.Documents.TableCell) in every row
at the corresponding index, keeping the table rectangular.
