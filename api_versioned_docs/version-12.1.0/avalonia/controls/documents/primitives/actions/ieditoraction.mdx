## Remarks

Editor actions provide a unified abstraction for operations like formatting,
clipboard operations, and undo/redo. They can be used across multiple UI surfaces:
toolbar buttons, context menus, and keyboard shortcuts.

Actions are typically singletons accessed via the [Avalonia.Controls.Documents.Primitives.Actions.EditorActions](xref:Avalonia.Controls.Documents.Primitives.Actions.EditorActions) class.
