## Remarks

This base class provides common functionality for actions that modify
properties on Block elements (Paragraph, Section, etc.).

When the selection spans multiple blocks, all affected blocks have their
property set. Override [Avalonia.Controls.Documents.Primitives.Actions.BlockPropertyAction&lt;T&gt;.ResolveTargetElements(Avalonia.Controls.Documents.Primitives.ITextEditorHost)](xref:Avalonia.Controls.Documents.Primitives.Actions.BlockPropertyAction%601.ResolveTargetElements%28Avalonia.Controls.Documents.Primitives.ITextEditorHost%29) to change the
set of elements that are modified (e.g. target [Avalonia.Controls.Documents.TableCell](xref:Avalonia.Controls.Documents.TableCell)
instead of [Avalonia.Controls.Documents.Paragraph](xref:Avalonia.Controls.Documents.Paragraph)).

Derived classes only need to:

- Specify the target StyledProperty
- Provide metadata (Id, DisplayName, Gesture)
- Return available values if applicable

### HasConsistentValue Method

Gets whether all affected blocks have the same value for this property.

```csharp
public bool HasConsistentValue(Avalonia.Controls.Documents.Primitives.ITextEditorHost host)
```

#### Parameters

`host` [Avalonia.Controls.Documents.Primitives.ITextEditorHost](xref:Avalonia.Controls.Documents.Primitives.ITextEditorHost)

The editor host.


#### Returns

bool

True if all affected blocks have the same property value.
False if blocks have different values (mixed state).


#### Remarks

UI controls should show a mixed/indeterminate state when this returns false.
For example, a [ComboBox](xref:Avalonia.Controls.ComboBox) might show "Mixed" placeholder text.

For collapsed selections, this always returns true (single block).
