## Remarks

When opened, the flyout computes the active [Avalonia.Controls.Documents.Primitives.Adorners.ToolbarTargetAreas](xref:Avalonia.Controls.Documents.Primitives.Adorners.ToolbarTargetAreas)
from the current block context and propagates them to each
[Avalonia.Controls.Documents.Primitives.Toolbar.EditorMenuItem](xref:Avalonia.Controls.Documents.Primitives.Toolbar.EditorMenuItem) so that items with non-matching
[Avalonia.Controls.Documents.Primitives.Toolbar.EditorMenuItem.TargetAreas](xref:Avalonia.Controls.Documents.Primitives.Toolbar.EditorMenuItem.TargetAreas) are hidden automatically.
Consecutive or trailing [Avalonia.Controls.Separator](xref:Avalonia.Controls.Separator) items left by hidden
groups are collapsed as well.
