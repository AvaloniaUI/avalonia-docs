## Remarks

[RichTextEditor](xref:Avalonia.Controls.RichTextEditor) is a full-featured text editing control that combines document rendering (TextViewBase)
with interactive editing capabilities (text selection, keyboard input, mouse input, etc.).

Implements [Avalonia.Controls.Documents.Primitives.ITextViewHost](xref:Avalonia.Controls.Documents.Primitives.ITextViewHost) for viewport/scrolling and [Avalonia.Controls.Documents.Primitives.ITextEditorHost](xref:Avalonia.Controls.Documents.Primitives.ITextEditorHost)
for editing component abstraction.

### ContentChanged Event

Occurs when the document content changes.

```csharp
public event EventHandler ContentChanged
```

#### Remarks

This event fires once per logical editing operation, not for each
individual character. Batch operations trigger a single event.

The event is only raised when the document was actually modified;
no-op change blocks (opened without edits) are suppressed.

</section>
