## Remarks

Note that depending on the type of grid being displayed, items retrieved from a
[Avalonia.Controls.TreeDataGridRows](xref:Avalonia.Controls.TreeDataGridRows) collection may be reused, so the [Avalonia.Controls.Models.ITreeDataGridRowModel](xref:Avalonia.Controls.Models.ITreeDataGridRowModel) should be
treated as valid only until the next item is retrieved from the collection or the layout of
the grid is updated.
