### ActualThemeVariant Property

Gets the UI theme that is currently used by the element, which might be different than the RequestedThemeVariantProperty.

```csharp
public Avalonia.Styling.ThemeVariant ActualThemeVariant { get; set; }
```

#### Returns

If current control is contained in the [ThemeVariantScope](xref:Avalonia.Controls.ThemeVariantScope), [TopLevel](xref:Avalonia.Controls.TopLevel) or Application with non-default RequestedThemeVariant, that value will be returned.
Otherwise, current OS theme variant is returned.
