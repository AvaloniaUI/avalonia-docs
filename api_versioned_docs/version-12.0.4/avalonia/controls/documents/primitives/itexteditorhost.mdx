## Remarks

Extends [Avalonia.Controls.Documents.Primitives.IInteractiveTextHost](xref:Avalonia.Controls.Documents.Primitives.IInteractiveTextHost) with editing-specific members:
read-only gating, return/tab acceptance. Components that perform text
mutation (typing, delete, paste, undo) require this interface.

Read-only interactive views should implement [Avalonia.Controls.Documents.Primitives.IInteractiveTextHost](xref:Avalonia.Controls.Documents.Primitives.IInteractiveTextHost)
directly instead.

#### Remarks

All callers should route through this method instead of calling
[Avalonia.Controls.Documents.Undo.IUndoManager.Redo](xref:Avalonia.Controls.Documents.Undo.IUndoManager.Redo) directly. The

```csharp
public event EventHandler IsReadOnlyChanged
```

#### Remarks

Toolbar tools and editor actions subscribe to this event to refresh
their enabled state when read-only mode is toggled at runtime.
The default implementation is a no-op, which is acceptable for
hosts that never toggle [Avalonia.Controls.Documents.Primitives.ITextEditorHost.IsReadOnly](xref:Avalonia.Controls.Documents.Primitives.ITextEditorHost.IsReadOnly).
