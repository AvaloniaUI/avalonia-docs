## Remarks

The [Avalonia.Controls.Documents.MarkdownImage.ImageLoader](xref:Avalonia.Controls.Documents.MarkdownImage.ImageLoader) is typically set via an Avalonia style,
mirroring how [Avalonia.Controls.Documents.MarkdownCodeBlock.Highlighter](xref:Avalonia.Controls.Documents.MarkdownCodeBlock.Highlighter) is configured:

```csharp
<Style Selector="MarkdownImage">
<Setter Property="ImageLoader" Value="{StaticResource myLoader}"/>
</Style>
```

Image loading is deferred until both [Avalonia.Controls.Documents.MarkdownImage.ImageSource](xref:Avalonia.Controls.Documents.MarkdownImage.ImageSource) and
[Avalonia.Controls.Documents.MarkdownImage.ImageLoader](xref:Avalonia.Controls.Documents.MarkdownImage.ImageLoader) are available. This avoids coupling the
snapshot renderer to the image loader.
