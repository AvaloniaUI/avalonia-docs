## Remarks

[TreeDataGridRowsPresenter](xref:Avalonia.Controls.Primitives.TreeDataGridRowsPresenter) is responsible for creating, recycling, and arranging rows
vertically in a [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid). It handles the virtualization of rows, ensuring that only
visible rows are realized in the visual tree to improve performance.

This presenter is typically used within a [Avalonia.Controls.TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid) control template to
display the rows for that control.
