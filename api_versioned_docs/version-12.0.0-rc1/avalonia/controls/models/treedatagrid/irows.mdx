## Remarks

Note that items retrieved from an [Avalonia.Controls.Models.TreeDataGrid.IRows](xref:Avalonia.Controls.Models.TreeDataGrid.IRows) collection may be reused, so the
[Avalonia.Controls.Models.TreeDataGrid.IRow](xref:Avalonia.Controls.Models.TreeDataGrid.IRow) should be treated as valid only until the next item is retrieved from
the collection.
