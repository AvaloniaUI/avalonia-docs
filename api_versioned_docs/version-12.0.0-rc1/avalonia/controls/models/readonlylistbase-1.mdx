## Remarks

This class should be considered internal to the [TreeDataGrid](xref:Avalonia.Controls.TreeDataGrid) package and is not intended
for use outside of this package.

Provides a base implementation for read-only list collections, implementing both
[System.Collections.Generic.IReadOnlyList&lt;T&gt;](xref:System.Collections.Generic.IReadOnlyList%601) and [System.Collections.IList](xref:System.Collections.IList) interfaces.
