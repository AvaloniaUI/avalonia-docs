#### Remarks

A style trigger is a selector such as .class which overrides a
[Avalonia.Data.BindingPriority.Template](xref:Avalonia.Data.BindingPriority.Template) value. In this way, a control can have, e.g. a Background from
the template which changes when the control has the :pointerover class.
