## Remarks

When an implementation of this interface is used as a drag source using [Avalonia.Input.DragDrop.DoDragDropAsync(Avalonia.Input.PointerEventArgs,Avalonia.Input.IDataTransfer,Avalonia.Input.DragDropEffects)](xref:Avalonia.Input.DragDrop.DoDragDropAsync%28Avalonia.Input.PointerEventArgs%2CAvalonia.Input.IDataTransfer%2CAvalonia.Input.DragDropEffects%29),
it must NOT be disposed by the caller. The system will dispose of it automatically when the drag operation completes.
