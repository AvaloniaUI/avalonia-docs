#### GetBaseValue Method

```csharp
public object GetBaseValue(Avalonia.AvaloniaObject target, Avalonia.AvaloniaProperty property)
```

##### Parameters

`target` [Avalonia.AvaloniaObject](xref:Avalonia.AvaloniaObject)

The object.


`property` [Avalonia.AvaloniaProperty](xref:Avalonia.AvaloniaProperty)

The property.


##### Returns

object


##### Remarks

For styled properties, gets the value of the property excluding animated values, otherwise
[Avalonia.AvaloniaProperty.UnsetValue](xref:Avalonia.AvaloniaProperty.UnsetValue). Note that this method does not return
property values that come from inherited or default values.

For direct properties returns the current value of the property.
