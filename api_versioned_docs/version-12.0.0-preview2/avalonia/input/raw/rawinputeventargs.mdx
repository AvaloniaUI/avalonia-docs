## Remarks

Raw input events are sent from the windowing subsystem to the [Avalonia.Input.InputManager](xref:Avalonia.Input.InputManager)
for processing: this gives an application the opportunity to pre-process the event. After
pre-processing they are consumed by the relevant [Avalonia.Input.Raw.RawInputEventArgs.Device](xref:Avalonia.Input.Raw.RawInputEventArgs.Device) and turned into
standard Avalonia events.
