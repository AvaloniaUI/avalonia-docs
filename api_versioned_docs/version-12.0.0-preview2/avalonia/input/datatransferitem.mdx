## Remarks

While it also implements [Avalonia.Input.IAsyncDataTransferItem](xref:Avalonia.Input.IAsyncDataTransferItem), this class always returns data synchronously.
For advanced usages, consider implementing [Avalonia.Input.IAsyncDataTransferItem](xref:Avalonia.Input.IAsyncDataTransferItem) directly.
