## Remarks

This class is used to define the geometry of a [Avalonia.Media.StreamGeometry](xref:Avalonia.Media.StreamGeometry). An instance
of [Avalonia.Media.StreamGeometryContext](xref:Avalonia.Media.StreamGeometryContext) is obtained by calling
[Avalonia.Media.StreamGeometry.Open](xref:Avalonia.Media.StreamGeometry.Open).
