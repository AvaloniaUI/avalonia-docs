### SetScrollPercent

Sets the horizontal and vertical scroll position as a percentage of the total content
area within the control.

```csharp
public System.Void SetScrollPercent(System.Double horizontalPercent, System.Double verticalPercent)
```

| Parameter | Description |
| --- | --- |
| `horizontalPercent` | The horizontal position as a percentage of the content area's total range.
[Avalonia.Automation.ScrollPatternIdentifiers.NoScroll](xref:Avalonia.Automation.ScrollPatternIdentifiers.NoScroll) should be passed in if the control
cannot be scrolled in this direction. |
| `verticalPercent` | The vertical position as a percentage of the content area's total range.
[Avalonia.Automation.ScrollPatternIdentifiers.NoScroll](xref:Avalonia.Automation.ScrollPatternIdentifiers.NoScroll) should be passed in if the control
cannot be scrolled in this direction. |
