## Remarks

The control class extends [Avalonia.Input.InputElement](xref:Avalonia.Input.InputElement) and adds the following features:

- A [Avalonia.Controls.Control.Tag](xref:Avalonia.Controls.Control.Tag) property to allow user-defined data to be attached to the control.
