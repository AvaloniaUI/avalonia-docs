## Remarks

This class acts as a base for top level windows such as [Avalonia.Controls.Window](xref:Avalonia.Controls.Window) and
[Avalonia.Controls.Primitives.PopupRoot](xref:Avalonia.Controls.Primitives.PopupRoot). It handles scheduling layout, styling and rendering as well as
tracking the window [Avalonia.Controls.TopLevel.ClientSize](xref:Avalonia.Controls.TopLevel.ClientSize) and [Avalonia.Controls.WindowBase.IsActive](xref:Avalonia.Controls.WindowBase.IsActive) state.
