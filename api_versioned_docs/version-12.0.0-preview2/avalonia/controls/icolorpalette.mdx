### GetColor Method

Gets a color in the palette by index.

```csharp
public Avalonia.Media.Color GetColor(int colorIndex, int shadeIndex)
```

#### Parameters

`colorIndex` int

The index of the color in the palette.
The index must be between zero and [Avalonia.Controls.IColorPalette.ColorCount](xref:Avalonia.Controls.IColorPalette.ColorCount).


`shadeIndex` int

The index of the color shade in the palette.
The index must be between zero and [Avalonia.Controls.IColorPalette.ShadeCount](xref:Avalonia.Controls.IColorPalette.ShadeCount).


#### Returns

[Avalonia.Media.Color](xref:Avalonia.Media.Color)

The color at the specified index or an exception.
