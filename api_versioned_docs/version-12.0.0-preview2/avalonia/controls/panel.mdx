## Remarks

Controls can be added to a [Avalonia.Controls.Panel](xref:Avalonia.Controls.Panel) by adding them to its [Avalonia.Controls.Panel.Children](xref:Avalonia.Controls.Panel.Children)
collection. All children are layed out to fill the panel.
