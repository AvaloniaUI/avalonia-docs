### OverlayDismissEventPassThrough Property

Gets or sets a value indicating whether the event that closes the flyout is passed
through to the parent window.

```csharp
public bool OverlayDismissEventPassThrough { get; set; }
```

#### Remarks

Clicks outside the popup cause the popup to close. When
[Avalonia.Controls.Primitives.PopupFlyoutBase.OverlayDismissEventPassThrough](xref:Avalonia.Controls.Primitives.PopupFlyoutBase.OverlayDismissEventPassThrough) is set to false, these clicks will be
handled by the popup and not be registered by the parent window. When set to true,
the events will be passed through to the parent window.

</section>
