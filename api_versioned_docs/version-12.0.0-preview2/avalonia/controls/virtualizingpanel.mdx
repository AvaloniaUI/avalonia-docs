## Remarks

Panels should implement the abstract members of this class to provide virtualization of
items in a [Avalonia.Controls.VirtualizingPanel.ItemsControl](xref:Avalonia.Controls.VirtualizingPanel.ItemsControl). Derived panels can manage scrolling by implementing
[Avalonia.Controls.Primitives.ILogicalScrollable](xref:Avalonia.Controls.Primitives.ILogicalScrollable) or by listening to the
[Avalonia.Layout.Layoutable.EffectiveViewportChanged](xref:Avalonia.Layout.Layoutable.EffectiveViewportChanged) event.

The methods on the [Avalonia.Controls.VirtualizingPanel.ItemContainerGenerator](xref:Avalonia.Controls.VirtualizingPanel.ItemContainerGenerator) should be used to create, prepare and
clear containers for items.
