### Initialized Event

Occurs when the styled element has finished initialization.

```csharp
public event EventHandler Initialized
```

#### Remarks

The Initialized event indicates that all property values on the styled element have been set.
When loading the styled element from markup, it occurs when
[System.ComponentModel.ISupportInitialize.EndInit](xref:System.ComponentModel.ISupportInitialize.EndInit) is called *and* the styled element
is attached to a rooted logical tree. When the styled element is created by code and
[System.ComponentModel.ISupportInitialize](xref:System.ComponentModel.ISupportInitialize) is not used, it is called when the styled element is attached
to the visual tree.

</section>
