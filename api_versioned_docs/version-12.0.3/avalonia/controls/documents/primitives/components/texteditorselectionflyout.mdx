## Remarks

The mini-bar opens on pointer release when a non-empty selection exists,
and closes automatically when the selection becomes empty or the user
presses Escape. The right-click context menu is handled separately by
[Avalonia.Controls.Documents.Primitives.Components.TextViewContextMenu](xref:Avalonia.Controls.Documents.Primitives.Components.TextViewContextMenu).
