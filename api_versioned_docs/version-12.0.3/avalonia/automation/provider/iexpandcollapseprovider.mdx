### ShowsMenu Property

Gets a value indicating whether expanding the element shows a menu of items to the user,
such as drop-down list.

```csharp
public bool ShowsMenu { get; set; }
```

#### Remarks

Used in OSX to allow `accessibilityPerformShowMenu` to open expandable controls such as a
[Avalonia.Controls.ComboBox](xref:Avalonia.Controls.ComboBox); in macOS, a combo box drop-down is considered a menu.

- WindowsNo mapping.
- macOS When true, `NSAccessibilityProtocol.accessibilityPerformShowMenu` will cause the [Avalonia.Automation.Provider.IExpandCollapseProvider.Expand](xref:Avalonia.Automation.Provider.IExpandCollapseProvider.Expand) method to be triggered.
