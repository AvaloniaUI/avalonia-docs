
#### Parameters

#### Returns

System.Threading.Tasks.Task&lt;object&gt;

A value for `format`, or null if the format is not supported.


#### Remarks

Implementations of this method are expected to return a value matching the exact type
of the generic argument of the underlying [Avalonia.Input.DataFormat&lt;T&gt;](xref:Avalonia.Input.DataFormat%601).
