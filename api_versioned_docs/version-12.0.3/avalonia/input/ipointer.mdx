### Capture Method

Captures pointer input to the specified control.

```csharp
public void Capture(Avalonia.Input.IInputElement control)
```

#### Parameters

`control` [Avalonia.Input.IInputElement](xref:Avalonia.Input.IInputElement)

The control.


#### Remarks

When an element captures the pointer, it receives pointer input whether the cursor is
within the control's bounds or not. The current pointer capture control is exposed
by the [Avalonia.Input.IPointer.Captured](xref:Avalonia.Input.IPointer.Captured) property.
