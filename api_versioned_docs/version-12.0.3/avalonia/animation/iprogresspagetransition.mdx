## Remarks

Transitions implementing this interface can be driven by a normalized progress value
(0.0 to 1.0) during swipe gestures or programmatic animations, rather than running
as a timed animation via [Avalonia.Animation.IPageTransition.Start(Avalonia.Visual,Avalonia.Visual,bool,System.Threading.CancellationToken)](xref:Avalonia.Animation.IPageTransition.Start%28Avalonia.Visual%2CAvalonia.Visual%2CSystem.Boolean%2CSystem.Threading.CancellationToken%29).
